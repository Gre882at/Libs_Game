 Libs_Game

A simple Python game that asks the user for words and generates a funny story.

## How to run
1. Make sure Python is installed.
2. Run the file in a terminal or code editor:
```
Libs.py
```

## How it works
- The game asks for a noun, verb, adjective, and place.
- Then it combines them into a short story.
- Shows the story in the terminal.
